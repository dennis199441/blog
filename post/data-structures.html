Data Structures

Programming is about the way we handle data

Posted by Dennis on November 5, 2017

According to Wikipedia, a data structure is a particular way of organizing and storing data in a computer so that it can be accessed and modified efficiently. More precisely, a data structure is a collection of data values, the relationships among them, and the functions or operations that can be applied to the data.

There are different kinds of data structure such as Array, List, Trees, etc. I am going to talk about three of them. They are Array, Linked list and Binary Tree.

Array

Array is the simplest data structure that consist of a collection of elements such as integer or string. Each element in the array is identified by an index. Arrays are used to implement other data structures, such as lists, stack, queue, ect.

Linked list

Linked list is a list that composed by serval nodes. Each node consists of two components, data and a pointer to the next node. The nodes are linked together by pointer and that why it is called linked list.

Unlike array, we don't use index to identify the node. Therefore, if we want to search a particular data in the linked list, we need to iterate through the list from the head node (using getNext() in my case).

An example of Linked List. An example of code of Node class in Java. An example of code of creating Linked List in Java.

Binary Tree

Binary Tree is, similar to linked list, composed by serval nodes. The difference is that each node consists of three components, data, left node and right node. The left child is linked to its parent by a pointer and so as the right child.

Similar to linked list, we don't use index to identify the nodes. Therefore, if we want to search a particular data in the tree, we need to use some algorithms such as Breadth First Search.
